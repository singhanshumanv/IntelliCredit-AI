# IntelliCredit AI

AI-powered credit risk analysis platform that analyzes financial documents and generates credit decisions.

## Features

- Document AI (RAG)
- Financial data extraction
- Risk scoring engine
- Credit decision engine
- Explainable AI
- External risk signals
- CAM report generation
- AI assistant

## Tech Stack

- Python
- Streamlit
- LangChain
- FAISS
- Groq LLM
- HuggingFace Embeddings

## Run Locally

pip install -r requirements.txt  
streamlit run app.py
